# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.1.1] - 2025-11-09

### Fixed
- Fixed a critical bug that was crashing ESLint due to incorrect use of `trim()` on an array

### Added
- Several new validation rules for enhanced Aabha decorator support

### Notes
- Recommended version for Aabha 1.1.0

## [1.1.0] - 2025-11-08

### Added
- Initial release of eslint-plugin-aabha
- Complete set of validation rules for Aabha decorators
- TypeScript support with full type safety
- Recommended configuration for quick setup
- Comprehensive documentation and examples
